Kebijakan Privasi

Kebijakan privasi ini berlaku untuk aplikasi HRPAY Kalkulator HRD.

1. Informasi yang Dikumpulkan

Aplikasi ini tidak mengumpulkan data pribadi pengguna secara langsung.

2. Penyimpanan Data

Aplikasi menyediakan fitur untuk menyimpan dan menghapus data yang dimasukkan oleh pengguna. Data tersebut disimpan secara lokal di perangkat pengguna dan tidak dikirim ke server manapun.

3. Iklan

Aplikasi ini menggunakan layanan iklan dari Google AdMob. AdMob dapat mengumpulkan data tertentu seperti ID perangkat dan aktivitas penggunaan untuk menampilkan iklan yang relevan.

Untuk informasi lebih lanjut, silakan lihat kebijakan privasi Google:

https://policies.google.com/privacy

4. Keamanan

Kami tidak menyimpan data pengguna di server, sehingga risiko kebocoran data sangat minimal.

5. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan di halaman ini.

6. Kontak

Jika ada pertanyaan, silakan hubungi melalui email yang terdaftar di halaman aplikasi.
